How IDOR Works Here
⚡
Vulnerable Mode: The server fetches any user by ID without checking who's asking. Alice (id=2) can request id=3 and get Bob's full data — including his password.
🛡
Fixed Mode: The server checks session.user.id === requested id. If they don't match (and you're not admin), it returns 403 Forbidden.
📋
Lesson: Always validate ownership server-side. Never trust the client to only send their own ID. URL parameters are trivially modified.
